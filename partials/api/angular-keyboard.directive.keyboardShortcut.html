keyboardShortcut
directive in module angular-keyboard

Description

Registers a keyboard shortcut for the default event for the attached element.

Use this directive to attach a keyboard shortcut to an element that already has some behavior attached to it: 

<a ng-click="doSomething()" keyboard-shortcut="ctrl+d">Do Something</a>

This will cause the ctrl+d shortcut to call doSomething().

There is a variaety of options to customize the behavior.

Usage

as element:
<keyboard-shortcut
       keyboard-shortcut="{string}"
       [keyboard-title="{string}"]
       [keyboard-category="{string}"]
       [keyboard-trigger="{string}"]
       [keyboard-action="{function}"]
       [keyboard-prevent-default="{boolean}"]
       [private-shortcut="{boolean}"]
       [shortcut-if="{boolean}"]
       [selection-shortcut="{boolean}"]>
</keyboard-shortcut>
as attribute
<ANY keyboard-shortcut="{string}"
     [keyboard-title="{string}"]
     [keyboard-category="{string}"]
     [keyboard-trigger="{string}"]
     [keyboard-action="{function}"]
     [keyboard-prevent-default="{boolean}"]
     [private-shortcut="{boolean}"]
     [shortcut-if="{boolean}"]
     [selection-shortcut="{boolean}"]>
   ...
</ANY>

Parameters

ParamTypeDetails
keyboardShortcutstring

The shortcut that the user needs to enter. See Keyboard shortcuts for more details.

keyboardTitle
(optional)
string

A description for the user of what the directive does. This is used in conjuction with the keyboardHelp directive. Also if the element does not specify a title, this with the keyboard shortcut will become its title. Defaults to an elements title or text.

keyboardCategory
(optional)
string

If used, this will group shortcuts into categories in the keyboardHelp directive.

keyboardTrigger
(optional)
string

Specify the event that should be run on the keyboard shortcut. Defaults to the click event. This means that attaching shortcuts to links and buttons should just work, but saving forms might need a different event.

keyboardAction
(optional)
function

Action to take when the keyboard shortcut is triggered. Defaults to triggering an event on the element. This allows to only expose functions through the HTML: 

<keyboard-shortcut keyboard-shortcut="mod+s" keyboard-action="save()" keyboard-title="Save the document">
</keyboard-shortcut>
keyboardPreventDefault
(optional)
boolean

Prevents the default keyboard action.

privateShortcut
(optional)
boolean

Hides the shortcut from the keyboardHelp directive and from exposure in KeyboardShortcuts.actions().

shortcutIf
(optional)
boolean

The keyboard shortcut is only active if the value of the expression is true.

selectionShortcut
(optional)
boolean

Set this if the shortcut should only apply if the element is selected via keyboardSelectable.
