KeyboardShortcuts
object in module angular-keyboard

Description

Centrally manages all active Keyboard shortcuts.

Methods

  • actions()

    Get all the public currently registered actions. Actions registered with the private flag set will not be included.

    Returns
    Array

    Returns an array of objects with keys coresponding to the parameters of register.

  • register(name, keybinding, action, options)

    Registers an action for a keyboard shortcut.

    Parameters
    ParamTypeDetails
    namestring

    A user visisble name that describes the action

    keybindingstring

    A keybinding string.

    actionfunction

    This function is called when the keybinding is invoked.

    options
    (optional)
    		Object

    An options object with the following boolean options:

    preventDefault will prevent the default behavior of the event. So if the action would be a mouse click on a link, the href wouldn't be followed.

    private This flag will hide the action from the keyboardHelp directive and from being exposed through the actions() method.

    digest This flag will run the action after the digest cycle completes rather than immediately. Useful when defining actions in custom directives and modifying scope variables in actions.

    Example

    KeyboardShortcuts.register('Access task manager', 'ctrl+alt+del', function () {
  TaskManager.show();
}, {
  private: true,
  digest: true
});

  • remove(name, keybinding)

    Un-registers an action from a keyboard shortcut.

    Parameters
    ParamTypeDetails

    keybindingstring
